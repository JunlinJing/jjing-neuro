这里展示了我参与的一些开源项目和软件工具，这些项目与我的研究兴趣和学术工作密切相关。这些工具可以供其他研究者使用， 也欢迎对这些项目提出反馈和贡献代码。

NeuroCNN

用于分析神经影像数据的卷积神经网络工具包。支持多模态数据融合，包括fMRI、DTI和结构性MRI数据的分析和分类。 该工具已应用于神经退行性疾病的早期诊断研究。

Python PyTorch 神经影像 深度学习

BrainML

为脑科学研究者设计的机器学习工具库，专注于处理多种脑信号数据（EEG、MEG、ECoG等）。 包括预处理、特征提取、分类和可视化等功能模块，支持实时数据流处理和离线分析。

Python Scikit-learn MNE 脑电分析

MetaCL

基于神经元元可塑性机制实现的持续学习框架。通过动态调整神经网络学习率和连接权重， 有效减轻灾难性遗忘问题。包括多种元学习算法实现和基准测试数据集。

Python TensorFlow 元学习 持续学习

AttentionVis

注意力机制可视化工具，可用于分析和展示人类在自然视觉场景中的注意力分配模式。 支持眼动数据分析、显著性图生成以及模型预测结果与真实数据的对比分析。

JavaScript D3.js Python 数据可视化

NeuroBCI

开源脑机接口开发框架，支持实时信号处理、解码和反馈。包括多种常用BCI范式（如P300拼写器、SSVEP、运动想象） 的实现，以及用于快速适应新用户的迁移学习模块。

C++ Python Qt 脑机接口
